# Summarize an `lmm` fit

Summarize an `lmm` fit

## Usage

``` r
# S3 method for class 'lmm'
summary(object, level = 0.95, ...)
```

## Arguments

- object:

  An `lmm` object.

- level:

  Confidence level for fixed effects.

- ...:

  Additional arguments.

## Value

An object of class `summary.lmm`.
