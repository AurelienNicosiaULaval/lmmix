# Repeated-measures growth example from the SAS MIXED documentation

Growth measurements for 11 girls and 16 boys at ages 8, 10, 12, and 14.
SAS uses these Potthoff and Roy data in Example 79.2 of the SAS/STAT
14.3 documentation for the MIXED procedure.

## Usage

``` r
sas_growth
```

## Format

A data frame with 108 rows and 4 variables:

- Person:

  Person identifier with 27 levels.

- Gender:

  Gender factor with levels F and M.

- Age:

  Age in years.

- y:

  Continuous growth response.

## Source

SAS Institute Inc. (2017). *SAS/STAT 14.3 User's Guide*, Example 79.2,
Repeated Measures.
<https://support.sas.com/documentation/onlinedoc/stat/>
